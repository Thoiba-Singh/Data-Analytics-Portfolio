# Retail Sales & Profit Dashboard
## Project Overview
Retail project analyzes a retail sales dataset to identify key business insights using Python and Power Bi.
## Tools Used
- Python (Pandas)
- Power Bi
## Project Workflow
1) Cleaned and explore the dataset using  python.
2) Created new columns:
   - Discount Amount
   - Selling Price
   - Proefit
3) Imported the cleaned dataset into Power Bi.
4) Build an interactive dashboard.
## Dashboard Features
- Total Profit
- Total Sales
- Total Quantity Sold
- Monthly Profit Trends
- Profit by Category
- Profit by Region
- Profit by Segment
- Top 10 Products by Profit
- Top 10 Cities by Profit
- Interactive slicers for Category, Region, Segment and Ship Mode
## Business Insights
- Consumers segment generated the highest profit.
- West region contributed the highest profit.
- Profit trends can be analyzed monthly.
- Top performing products and cities are easily identified.
