# HR Analytics Dashboard
## Project Overview
This project analyzis HR data using Python for data exploration and Power Bi for building an interactive HR analytics dashboard. The Dashboard helps understand employee attrition, workforce demographics, and key HR metrices.
## Tools Used
- Python
- Pandas
- Matplotlib
- Power Bi
## Key Insights
- Analyzed employee attrition trends
- Compared attrition across departments
- Identified attrition by age group and gender
- Visualized employee  distribution by job role
- Monitored overall HR KPIs
