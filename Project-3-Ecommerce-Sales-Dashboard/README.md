# E-Commerce Sales Dashboard
## Project Overview
This project analyzes e-commerce sales data using SQL for data querying, Python for data exploration, and Power Bi for building an interactive dashboard. The dashboard provides insights into salesperformance, customer behaviour, and business trends
##Tools Used
- SQL
- Power Bi
